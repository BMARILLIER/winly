# Winly — Roadmap

## Principe directeur
Winly doit etre adapte a **plusieurs plateformes** (Instagram, TikTok, Twitter/X, LinkedIn, YouTube) — chaque fonctionnalite doit penser multi-plateforme des la conception.

---

## Priorite 1 — Fondations (donnees reelles)

- [ ] **Connecter Instagram** — OAuth Meta, sync posts/stats/engagement
- [ ] **Remplacer les donnees mock** — Growth Score, opportunites, signaux trending bases sur de vraies donnees
- [ ] **Import CSV/manuel des stats** — alternative si config Meta trop penible (export depuis l'app Instagram)

## Priorite 2 — Contenu & Creation

- [ ] **Templates visuels** — modeles de carrousels/stories pre-formates a remplir avec le texte genere
- [ ] **Banque de hooks intelligente** — Claude analyse les hooks passes, identifie ceux qui marchent, en genere de similaires
- [ ] **Detecteur de tendances** — scraper hashtags trending par niche, proposer des sujets
- [ ] **Voice-to-content** — parler, Claude transcrit et transforme en post
- [ ] **A/B testing de hooks** — proposer 2 versions, comparer les performances
- [ ] **Analyse de photos** — uploader 2-5 photos, Claude analyse et recommande la meilleure a poster

## Priorite 3 — Planification & Publication

- [ ] **Calendrier editorial visuel** — drag & drop, vue semaine/mois
- [ ] **Rappels de publication** — notifications quand c'est l'heure de poster
- [ ] **Publication directe** — publier sur Instagram/TikTok/Twitter depuis Winly via les APIs
- [ ] **Meilleur moment pour poster** — base sur les stats d'audience (quand elle est active)
- [ ] **Score de regularite** — tracker la frequence de publication, alerter quand ca decroche

## Priorite 4 — Engagement & Croissance

- [ ] **Dashboard concurrents** — suivre 3-5 comptes, voir ce qui marche, detecter les tendances
- [ ] **Simulateur de croissance** — projections basees sur la frequence de post
- [ ] **Multi-plateforme** — adapter chaque fonctionnalite a Instagram, TikTok, Twitter/X, LinkedIn, YouTube

## Priorite 5 — Monetisation

- [ ] **Media kit generator** — PDF pro avec stats pour demarcher des marques
- [ ] **Calculateur de tarifs** — combien facturer un partenariat selon la taille d'audience
- [ ] **Detecteur d'opportunites sponsoring** — marques qui matchent la niche

## Priorite 6 — UX / Qualite de vie

- [ ] **Mode sombre / clair** toggle
- [ ] **App mobile (PWA)** — utiliser Winly depuis le telephone
- [ ] **Notifications push** — rappels, alertes trending
- [ ] **Multi-workspace** — gerer plusieurs comptes depuis un seul login
